## QuickType
A Tool for Premiere Pro that lets you add Subtitles Easily

# Shortcuts
-
Ctrl + E (^e) — Set & Click Coordinates

What it does: Activates a "set" mode. The very next time you click your left mouse button, the script remembers that exact screen position, double-clicks it, switches focus back to your Premiere Timeline (Shift + 3), and deselects everything (Ctrl + Shift + A).

-
Ctrl + T (^t) — Create & Select Text

What it does: Creates a new text layer, centers the text alignment, double-clicks the custom coordinates you saved with Ctrl + E, and selects all text so you can start typing immediately.

-
Ctrl + G (^g) — Save & Return to Timeline

What it does: Exits text editing mode (Esc), shifts your workspace focus back to the Timeline panel, and deselects all clips.

-
Ctrl + K (^k) — Emergency Panic Button

What it does: Immediately shuts down this AutoHotkey script from running completely (works globally, even if Premiere is minimized).
